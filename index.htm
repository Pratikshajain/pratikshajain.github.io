IWAMS

IWAMS is designed to perform necessary analysis for planning watershed management projects through the following three modules:

  1. Hydrological analysis module : which includes Process based soil erosion analysis

  2. Prioritization module to identify critical area based on hydrology as well as poverty aspects

  3. Conservation module for optimal selection and allocation of soil and water conservation measures

IWAMS plugin selection option

The plugin allow to apply the methodology and main recommendations included (mainly) in these papers:

  • Jain, P. and Ramsankaran, RAAJ., 2018. GIS-based modeling of soil erosion processes using the Modified Morgan-Morgan Finney model (M-MMF) model in a large watershed having vast agro-climatological differences. Earth surface processes and landforms, 43(10), pp. 2064-2076. https://doi.org/10.1002/esp.4372

  • Jain, P. and Ramsankaran, RAAJ., 2019. GIS-based integrated multi-criteria modeling framework for watershed prioritization in India- A demonstration in marol watershed”. Journal of hydrology, 578 (124131). https://doi.org/10.1016/j.jhydrol.2019.124131

Hydrological Analysis Module

The hydrological analysis module integrates the Modified Morgan-Morgan Finney (M-MMF)(Morgan and Duzant, 2008) model to perform process-based soil erosion modeling. Hydrological analysis module is divided and ordered by three modules:

  1. Simple run
  2. Sensitivity analysis
  3. Calibration
  4. Validation
